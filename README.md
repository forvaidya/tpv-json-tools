# Schema Extractor

A utility to extract schema and save it to a file.

## Prerequisite 
```pip3 install genson```

## Usage 
```python3 diff_schema.py file_1.json file2.json```

## Output 
```file_1_output_schema.json``` and ```file_2_output_schema.json```

Use VS.Code to see the diff.
